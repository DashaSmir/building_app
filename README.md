# Building-App
Консольное приложение, которое позволяет сформировать заявку на строительные материалы с учетом региона, групп товаров и логики "удержания клиента".

## Функциональность
- Выбор региона из трёх доступных: Санкт-Петербург, Москва, Краснодар.
- Отображение каталога материалов с актуальными ценами для выбранного региона.
- Формирование заказа с сохранением в JSON-файл.
- Логика удержания клиента при отказе:
  - если выбранный товар не самый дешёвый в своей категории — предлагается более дешёвый аналог;
  - если товар уже самый дешёвый — предлагается скидка 5%.
- Возможность вернуться к выбору товара или региону на любом этапе
## Технологии
- **Node.js** (основная логика)
- **PostgreSQL** (хранение каталога товаров)
- **Docker / Docker Compose** (контейнеризация)
- **Inquirer.js** (интерактивный консольный интерфейс)
- **Jest** (unit-тестирование)


## Использование
Перед использованием необходимо скачать Docker Desktop и запустить его
```bash
# Запуск приложения (поднимет БД и откроет консольный интерфейс)
npm start

# Остановка всех контейнеров
npm stop

# Сборка образа после изменений в коде
npm run build

# Прогон тестов
npm test
```
Все команды выполняются из корня проекта
